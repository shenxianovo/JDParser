职位描述
1. 智能体系统架构设计与开发：设计并实现教育场景的 AI Agent 后端服务架构，支持任务规划、工具调用、多步推理等核心能力,构建高可用的智能体运行时环境
2. 上下文工程与记忆系统：设计 Agent 的上下文管理机制（Context Engineering）和记忆系统，结合向量数据库（Milvus/Pinecone）实现高效检索，优化上下文窗口利用效率
3. MCP 协议与 Skill 集成：基于 MCP（Model Context Protocol）等标准协议设计工具调用层，构建可复用的 Skill 能力库，集成内外部 API、数据库等资源
4. 后端服务开发与 API 设计：开发高性能 RESTful/gRPC API，支持流式传输（Streaming）、异步任务处理，与现有业务系统进行集成对接
5. 稳定性与可观测性保障：建立监控、日志、追踪体系（Prometheus/Grafana），针对大模型调用异常设计降级策略，保障系统 SLA
6. 技术跟进与创新：快速跟进 AI Agent 领域最新技术（Agent Skills、上下文工程、多智能体协作等），探索新技术在业务场景的落地应用
