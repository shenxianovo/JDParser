职位描述
1、负责音乐智能助手的 AI Agent 系统开发，包括意图识别、对话管理、任务规划等核心模块的设计与实现；
2、开发基于大语言模型的音乐推荐对话能力，实现自然语言驱动的歌曲搜索、歌单生成、风格探索等功能；
3、构建 Agent 的记忆系统（短期对话记忆、长期偏好记忆），实现个性化的上下文理解与用户画像积累；
4、参与情绪感知模块的开发，结合用户行为数据（播放、跳过、收藏等）和上下文信号（时间、场景）进行情绪状态推断；
5、开发 Agent 的工具调用能力，对接内部推荐系统、搜索服务、内容平台等，实现推荐→播放→反馈的端到端闭环；
6、参与 Prompt Engineering 设计与优化，支撑人设系统和情感化回复的生成质量；
7、关注大模型推理延迟优化，保障短交互场景下的响应速度和用户体验；
8、跟踪 AI Agent 领域最新技术进展，快速验证并推动新技术在业务中的落地。
职位要求
1、本科及以上学历，计算机科学、软件工程、人工智能等相关专业，2 年以上后端开发经验；
2、精通 Python 或 Java/Go 中的至少一门语言，具备扎实的数据结构、算法基础和良好的工程素养；
3、熟悉大语言模型（LLM）的原理与应用，有实际的 Prompt Engineering、RAG 或 Agent 开发经验；
4、熟悉至少一种主流 Agent 开发框架或 SDK（如 LangChain、LangGraph、Claude Agent SDK、AutoGen 等），有实际项目经验；
5、熟悉 Agent 核心机制，包括 Function Calling / Tool Use、ReAct、记忆管理、多轮对话上下文控制等；
6、熟悉常用中间件和数据库（MySQL、Redis、Kafka 等），具备高并发服务开发经验者优先；
7、具备较强的学习能力和技术敏感度，能快速跟进大模型和 Agent 领域的技术迭代；
8、良好的沟通协作能力，能与产品、算法、前端团队高效配合，推动需求从方案到上线的全流程。

加分项：
1、有音乐、内容推荐、搜索等相关业务开发经验；
2、有对话系统、情感计算、NLP 相关项目经验；
3、有向量数据库（Milvus、Chroma 等）或知识库构建经验；
4、有大模型微调（SFT/LoRA）或推理优化经验；
5、有个人 AI 项目、开源贡献或技术博客输出。
